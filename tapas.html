DISEÑO DE TAPAS — Trilogía

Diseño de portadas para una trilogía de novelas de ficción inspiradas en hechos reales, junto con el desarrollo de las primeras páginas de uno de los libros. Las tapas fueron diseñadas para captar la esencia única de cada historia, utilizando elementos visuales que reflejen el tono y la temática de cada novela, asegurando cohesión visual entre las tres pero manteniendo la individualidad de cada una.
